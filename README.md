zzdwm
=====

## URLs

    ## https://dwm.suckless.org/
    ## https://dl.suckless.org/dwm/
    ## https://dl.suckless.org/dwm/dwm-6.5.tar.gz

## Build dependence

    $ sudo apt install libx11-dev libxft-dev libxinerama-dev
    $ sudo apt install libasound2-dev

## Build dwm source

    $ cd src
    $ make
    $ sudo make install

## Install dwm session

    $ ./install.sh
    $ ./install.sh all
